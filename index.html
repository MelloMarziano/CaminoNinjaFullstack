Introducción

En este pequeño libro/guía aprenderás los fundamentos y buenas prácticas del desarrollo web moderno. Veremos HTML y CSS, JavaScript y TypeScript, usaremos Bootstrap 5 para construir interfaces responsivas y crearemos APIs con Node.js integrando PostgreSQL.
